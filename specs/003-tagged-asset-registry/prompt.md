# Prompt original

> Grouper les skills / commands / rules par module thématique : common, Typescript/javascript, python.
> Comment ? => tenir un fichier .yml de clé (le path vers le skill / commands / rules / hooks) vers un tableau de string typé (les tags).
> Certains items peuvent être dans plusieurs catégories / tag.
> Prévois un JSON Schema pour ce yml, pour que la structure soit propre, les tags possibles centralisés, et l'évolutivité du format tracée.
>
> L'objectif suivant sera de pouvoir récupérer l'ensemble des commandes / rules / skills etc… par thème.
> Parcours les différents projets sur cette machine pour trouver ces thèmes.
