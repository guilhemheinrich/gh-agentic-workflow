# Bitbucket Pipelines - Get Started

**Source**: [Atlassian Support](https://support.atlassian.com/bitbucket-cloud/docs/get-started-with-bitbucket-pipelines/)

Bitbucket Pipelines is an integrated CI/CD service built into Bitbucket Cloud. It allows you to automatically build, test, and deploy your code based on a configuration file in your repository.

Essentially, pipelines create containers in the cloud for you. Inside these containers, you can run commands (like you might on a local machine) but with all the advantages of a fresh system.

A pipeline is defined using a YAML file called `bitbucket-pipelines.yml`, located at the root of your repository.
