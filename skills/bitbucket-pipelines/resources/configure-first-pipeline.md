# Configure Your First Pipeline

**Source**: [Atlassian Support](https://support.atlassian.com/bitbucket-cloud/docs/configure-your-first-pipeline/)

## Prerequisites

- Bitbucket Cloud account
- At least one repository in your workspace

## Steps

1. In Bitbucket, go to your repository and select **Pipelines**
2. Select **Create your first pipeline** to scroll to the template section
3. Choose one of the available templates (RECOMMENDED if unsure)

Templates cover apps, microservices, mobile IaC, serverless development. Supported cloud providers include AWS, Azure, and GCP.

4. Once you choose a template, you'll land in the YAML editor to configure your pipeline

## Options

- **Change template**: Open dropdown and select different template (overrides existing)
- **Add more steps**: Copy code snippet from steps panel options
- **Add pipes**: See [What are pipes](pipes.md) for third-party tool configuration
- **Add variables**: Define custom variables for YAML file usage

See [Variables and secrets](variables-secrets.md) for more details.

## Outcome

Configure your first pipeline and always access settings via the pipeline cog icon.
